在学术界，arXiv 的预印本系统解决了一个关键问题：证明你的研究想法在某个时间点已经存在。但对于非学术领域的个人思考者来说，我们缺少一个类似的轻量级工具。

## 问题的本质

当你在社交媒体上分享一个想法时，平台可以随时修改或删除记录。你无法真正"拥有"自己思想的时间证明。

## Git 的优雅解决方案

Git 的提交记录天然具备了我们需要的特性：

- **不可篡改性**：每个 commit 的 SHA 哈希是基于内容计算的
- **时间戳**：精确到秒的提交时间
- **公开可验证**：托管在 GitHub 上，任何人都能查看
- **免费**：不需要任何额外费用

这不如区块链那样去中心化，但对于个人博客来说，已经足够了。

## 实际意义

$P(\text{idea\_proven}) = 1 - (1 - p)^n$

其中 $p$ 是单次验证的可信度，$n$ 是独立验证源的数量。GitHub 作为一个大型第三方平台，提供了足够高的 $p$ 值。
